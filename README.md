# SCORA
FC SCORA is an anime-styled FIFA World Cup match tracker. Fetches live scores from the football-data.org API with a fallback to mock data. Built with Node.js, Express, HTML, CSS &amp; vanilla JS.
**Features:**
- Anime/manga themed UI with speedlines and impact overlays
- Live, upcoming, and completed match filters
- Real-time goal simulation mode
- Match detail modal with timeline
